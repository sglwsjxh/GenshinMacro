# 原神鼠标宏工具

原神鼠标宏工具 — 统一的 Tkinter GUI 管理两个鼠标宏，通过鼠标侧键触发。

## 功能

- **X（鼠标侧键下键）** — 按住持续旋转视角（那维莱特重击旋转）
- **X2（鼠标侧键上键）** — 按住重复执行双击右键序列

## 安装

1. 确保已有 Python 3.10+
2. 创建并激活虚拟环境：

```bash
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
```

## 运行

```bash
python main.py
```

启动后会出现 GUI 窗口，点击「启动」按钮激活两个宏。点击「停止」按钮或关闭窗口即可退出。

## 快捷键

| 按键 | 功能 |
|------|------|
| X（侧键下键） | 按住 → 持续旋转视角 |
| X2（侧键上键） | 按住 → 重复双击右键 |
| Esc | 紧急停止并退出 |

## 注意事项

- **管理员权限**：首次启动时会自动请求管理员权限（因为 `pydirectinput` 需要硬件级输入权限）。
- **游戏内生效**：本工具使用 `pydirectinput` 和 `pyautogui` 模拟鼠标输入，请确保在游戏中运行时游戏窗口处于前台。
- **退出方式**：点击 GUI 的「停止」按钮、关闭窗口、或按 `Esc` 键均可安全退出。

## 依赖

- `pyautogui` — 鼠标控制
- `pydirectinput` — 游戏内鼠标移动
- `keyboard` — 鼠标侧键检测

## 已知问题

- `keyboard.mouse` 模块已废弃，未来版本可能替换为独立的 `mouse` 包。
